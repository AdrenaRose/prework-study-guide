HTML

  • The head element contains information about the webpage. This information is not visible to the user and is called metadata.
  • Metadata tags include:
    • <"title">
    • <"style">
    • <"base">
    • <"link">
  • The body element contains the content that we want the user to see.
  • There may be up to 6 heading <"h1,h2,h3,etc"> elements in the body of one HTML document.
  • A child element is one that is nested within other elements. Those other elements are called parent elements.
  • Semantic HTML elements are those that use the name of the element to describe its contents.
