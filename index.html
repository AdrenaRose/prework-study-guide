HTML

  • The head element contains information about the webpage. This information is not visible to the user and is called metadata.
  • Metadata tags include:
    • <"title">
    • <"style">
    • <"base">
    • <"link">
  • The body element contains the content that we want the user to see.
  • There may be up to 6 heading <"h1,h2,h3,etc"> elements in the body of one HTML document.
  • A child element is one that is nested within other elements. Those other elements are called parent elements.
  • Semantic HTML elements are those that use the name of the element to describe its contents.

CSS

  • CSS = Cascading Style Sheets
  • Defines colors, aesthestics, visual layout, and organization of a webpage
  • Inline CSS is used when a developer wants to style an element directly in the HTML file
    • Useful for quick and permanent changes
    • Applied to a specific HTML tag, using a style attribute with a CSS rule
    • Each inline style must be edited individually whenever there is a design change
    • Example: [h1 style="color:red;"] Prework Study Guide[/h1]
  • Internal CSS style sheet
    • CSS style sheet embedded directly in the HTML file
    • Use a [style] element in the [head] element
    • Rules apply to that page only
    • Classes and ids are used to identify elements and style multiple elements in the page code
  • External CSS style sheet
    • Most common way to style a webpage
    • Preferred because it allows developers to keep all their CSS rules in a separate file, making design changes easier
    • To use, the external style sheet is linked in the HTML's head element using a link element
  • Breakdown of a CSS rule: div (selector) {color: (property) blue (value); (declaration) text-align: (property) justify (value); (declaration)}
    • Selector defines the element or attributes to which the declarations will apply
    • Declarations contain the CSS property we want to apply and the value of that property
    • Two or more declarations in a row is called a declaration block
  • The margin property dictates how much space we want around the *outside* of an element.
  • The padding property adds space around the content *inside* an element. If padding is increased, the size of the entire element is increased.
  • Consider every HTML element like a box
    • CSS box model - each box (element) consists of margins outside the element, and the element's borders, padding, and content.
    • Content, padding, and border are *inside* the element.
  • The * (wild card symbol) can be used as a selector to apply a rule to ALL elements on a page.
  • Two selectors can be combined into a single CSS rule with a comma
  • Class attributes are used when we would like to apply styling to certain elements, but not all elements of the same type
  • Class attributes are then used as CSS selectors so that we can assign properties to them
